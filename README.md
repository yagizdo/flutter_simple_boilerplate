# Flutter - Boilerplate project template


## Run Locally

Clone the project

```bash
  git clone https://github.com/yagizdo/flutter_simple_boilerplate
```

Go to the project directory

```bash
  cd flutter_simple_boilerplate
```

Install dependencies

```bash
  flutter pub get
```

Run App

```bash
  flutter run
```
